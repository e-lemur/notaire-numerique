Trois principes, une garantie : l'intégrité.

Zero-knowledge par défaut

Le contenu de vos fichiers ne quitte jamais votre appareil. Seule l'empreinte SHA-256 (64 caractères) est transmise. Le secret professionnel est préservé par construction, pas par promesse contractuelle.

Antériorité incontestable

Chaque scellement est inscrit dans une hash-chain append-only. Une modification d'un seul octet du document original produit une empreinte différente et invalide automatiquement le certificat. Aucune réécriture du registre n'est possible.

Vérification publique, par tous

N'importe quel tiers — juge, banque, confrère, assurance — peut vérifier l'authenticité d'un document depuis son téléphone en scannant le QR code du certificat. Aucun compte, aucune installation requise côté vérificateur.

Comment ça marche

Trois clics, dix secondes — et vous avez un certificat opposable.

  1. 01

    Sélectionnez votre fichier

    Glissez-déposez n'importe quel document — acte authentique, contrat, testament, conclusion, ordonnance, PV. Le format ne change rien.

  2. 02

    Le hash est calculé localement

    Votre navigateur calcule l'empreinte SHA-256 du fichier puis envoie uniquement ces 64 caractères à notre registre. Le contenu reste sur votre machine.

  3. 03

    Recevez votre certificat

    Téléchargez immédiatement un PDF Certificat de Fiducité Numérique horodaté avec QR code de vérification. Annexable à toute procédure.
